## Setup

1. `pnpm create next-app -e https://github.com/noelrohi/next15-starter.git my-app`
2. `cd my-app`
3. `cp .env.example .env`
4. `pnpm install`
5. `pnpm run dev`

## Technologies

- [Next.js](https://nextjs.org)
- [Drizzle](https://orm.drizzle.team/docs/getting-started/introduction)
